# 🎥 Enregistreur d'Écran en Arrière-plan

## 🚀 Re-Exécution du Script

Pour utiliser un enregistreur d'écran et sauvegarder vos captures, suivez ces étapes :

---

## 📂 1. Créez le répertoire pour les captures

Avant de lancer le script, créez un répertoire où les captures seront sauvegardées :

```bash
mkdir captures
```

---

## ▶️ 2. Lancez le script

Exécutez le script avec la commande suivante :

```bash
python3 screenshot.py
```

Les captures seront sauvegardées dans le répertoire `captures`.
```
